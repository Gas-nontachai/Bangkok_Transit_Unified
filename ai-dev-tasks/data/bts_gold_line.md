# BTS Gold Line – Detailed Line, Stations, and Fare Structure
อัปเดตข้อมูลโครงสร้างสายและอัตราค่าโดยสารปกติ (ผู้ใหญ่)

หมายเหตุสำคัญ:
- สายสีทองเป็นระบบรถไฟฟ้าแบบ APM เชื่อมต่อจากสถานีกรุงธนบุรี
- ใช้อัตราค่าโดยสารแบบราคาเดียว (Flat Fare) ตลอดสาย
- ค่าโดยสารด้านล่างเป็นอัตราปกติ (Normal Adult Fare) ตามโครงสร้างประกาศ ไม่รวมส่วนลดหรือแคมเปญเฉพาะช่วงเวลา

---

# 1) BTS สายสีทอง (Gold Line)

จำนวนสถานี: 3 สถานี  
ประเภท: APM (Automated People Mover)  
เส้นทาง: กรุงธนบุรี -> คลองสาน

## รายชื่อสถานีทั้งหมด (3)

1. กรุงธนบุรี  
2. เจริญนคร  
3. คลองสาน  

## โครงสร้างค่าโดยสาร

- ค่าโดยสารตลอดสายแบบราคาเดียว: 16 บาท

โครงสร้าง:
- ไม่คิดตามจำนวนสถานี
- ค่าโดยสารเท่ากันทุกระยะภายในสายสีทอง

# หมายเหตุด้านความถูกต้องของข้อมูล

- รายชื่อสถานีอ้างอิงจากแนวเส้นทางที่เปิดใช้งานจริง
- ค่าโดยสารอ้างอิงโครงสร้างราคาเดียวตามประกาศใช้งานของสายสีทอง
- หากมีประกาศปรับราคาในอนาคต ควรอัปเดตไฟล์นี้ให้ตรงกับประกาศล่าสุดของผู้ให้บริการ
